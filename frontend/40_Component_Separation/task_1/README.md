## Atomic Design

Web ページという言葉は、本と同様に、ページが均一的で孤立しており、定量化可能であるという間違った仮定を表している。

実際には Web ページは、インタラクティブであり、動的なものであり、本のように均一的なものではなく、ページを構成する機能やコンポーネントによって左右されている。
