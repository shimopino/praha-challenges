## 環境設定

以下のコマンドでプロジェクトを初期化する。

```bash
# Next.js アプリケーション (v12) の初期化
npx create-next-app@latest --typescript

# Storybook の導入
npx sb init

# アプリけーションの起動
yarn dev

# Storybookの起動
yarn storybook
```
